# Images Directory

## Ministry of Health Logo

Please place the official Ministry of Health logo file here as `moh-logo.png`.

The logo should be:
- Circular format (or will be displayed in a circular frame)
- High resolution (recommended: 200x200px or larger)
- PNG format with transparent background preferred
- File name: `moh-logo.png`

The logo will be displayed on the login page (`login.html`).
